# canned_fish
   欢迎使用图片裁切（摸鱼）软件“鱼罐头”，以下是“鱼罐头”的功能以及操作指南：

## 功能
  将选定输入文件夹中所有的png文件通过旋转裁剪的方式转化成总像素点最少的矩形图片，并输出到指定的输出文件夹中。

## 操作：
### 选择输入输出文件夹
  可以通过点击按钮的方式进行选择，也可以将目标文件夹直接拖拽到对应的文字框中。
### 点击开始转化
  如果输入的文件夹信息有误，系统会弹出警告并阻止转化。
### 等待转化
  在转化期间可以点击“取消”中止转化，已经转化的图片会放到目标文件夹中。
### 转化完成
  此界面可以选择打开转化后的文件夹。
